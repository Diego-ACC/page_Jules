Featured Projects:

Web Application for "Mexicanos Primero Jalisco"

To support the mission of Mexicanos Primero Jalisco, I developed a web-based demo designed to bridge the gap between private donors and schools in need of educational resources. Tasked with creating a robust system for managing these connections, I architected a backend using JavaScript, Node.js, and Knex.js to interface with a PostgreSQL database. I focused with a teammate on building a scalable relational schema that could accurately track donor contributions and school requirements. Although the project remains a high-fidelity prototype, it successfully demonstrated to the association how a data-driven platform could streamline the donation process and improve educational infrastructure across the state.

Braille to Text

During an intensive one-week development sprint focused on image processing, I collaborated on a Python application designed to translate Braille into digital text. The primary challenge was developing a system that could accurately identify Braille patterns from various image sources within a very tight deadline. My partner and I implemented a model using matrix-based detection to isolate characters, while I worked on the preprocessing logic to handle variations in image quality. We achieved high translation accuracy during the final demonstration, proving the viability of using standard Python libraries to create functional accessibility tools in a short timeframe.

Portfolio Web

Recognizing that industry trust is built on more than just code, I developed a personal web portfolio to showcase both my technical skills and my personality as a developer. Following advice from a recruiter at Oracle, I designed the site to serve as a comprehensive "digital identity" that goes beyond a traditional resume. Simple Videogame (Metroidvania)

Driven by an interest in complex game logic and "Metroidvania" style progression, I developed a 2D platformer in Unity inspired by classics like Castlevania. While I enjoyed the creative process of level design and character mechanics, the core technical challenge lay in developing sophisticated enemy AI behaviors. I implemented state machines in C# to manage patrol, chase, and combat patterns, ensuring the gameplay felt reactive and challenging. The final result was a playable demo that showcased my ability to manage real-time game states and solve complex logic problems within a popular game engine.
